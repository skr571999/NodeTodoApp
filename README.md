# Node Todo App

## Technologies and Tools

- TextEditor - Visual Studio Code
- VSCode Extensions
  - Prettier - for code formatting
- Development OS - Window 10
- Frontend - ejs(html)
- Backend - NodeJS(ExpressJS)

## Usage

```cmd
git clone https://github.com/skr571999/NodeTodoApp.git

npm install
npm start

```
